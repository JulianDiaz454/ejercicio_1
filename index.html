Las etiquetas en HTML


¿Que son las etiquetas?

, las etiquetas son los bloques de construcción fundamentales que estructuran y definen el contenido de una página web. Funcionan como marcadores que le indican al navegador cómo debe interpretar y mostrar la información.


Etiquetas


Etiqueta Descripción Atributos Comunes
< a > Crea un hipervínculo a otras páginas web, archivos o ubicaciones dentro de la misma página. href, Target, Rel
< div > Se usa para definir un parte del html, se pueden agrupar contenido y etiquetas. id class style
< img > Permite colocar imagenes en la pagina. src, alt, width, height
< form > Permite crear un formulario para conseguir infromación del usuario action, method
< input > Crea campos de entreda dentro del formulario para ingresar los datos. type, name, valuea
